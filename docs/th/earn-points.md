# รับคะแนน (Earn Points)

## ภาพรวม

ระบบคะแนน YC365 ออกแบบมาเพื่อตอบแทนผู้ใช้ที่มีส่วนร่วมและมีส่วนช่วยในการเติบโตของแพลตฟอร์ม คะแนนเหล่านี้ไม่ได้เป็นเพียงตัวเลข แต่เป็นตัวแทนของการมีส่วนร่วมของคุณในระบบนิเวศของเราและสามารถนำไปสู่ผลประโยชน์ที่จับต้องได้ รวมถึงสิทธิ์ในการได้รับ Airdrop ในอนาคต

## วิธีการรับคะแนน

คุณสามารถสะสมคะแนนได้ผ่านกิจกรรมต่างๆ บนแพลตฟอร์ม:

### 1. การเชิญเพื่อน (Referral Program)

แบ่งปัน YC365 กับเครือข่ายของคุณและรับรางวัลด้วยกัน!

- **รางวัลผู้เชิญ**: รับ **1,000 คะแนน** สำหรับเพื่อนทุกคนที่คุณเชิญสำเร็จ
- **รางวัลผู้ถูกเชิญ**: เพื่อนของคุณจะได้รับ **1,000 คะแนน** เมื่อสมัครผ่านลิงก์ของคุณ
- **โบนัสเพิ่มเติม**: รับเปอร์เซ็นต์ของคะแนนที่เพื่อนของคุณได้รับ (เร็วๆ นี้)

!!! tip "เคล็ดลับ"
    ยิ่งคุณเชิญเพื่อนมากเท่าไหร่ คุณก็ยิ่งได้รับคะแนนมากเท่านั้น! ไม่มีขีดจำกัดจำนวนเพื่อนที่คุณสามารถเชิญได้

### 2. การฝากเงิน (Deposits)

เติมเงินในบัญชีของคุณเพื่อเริ่มการซื้อขายและรับคะแนน

- **อัตราการรับคะแนน**: รับ **1,000 คะแนน** สำหรับทุกๆ **$1,000** ที่คุณฝาก
- **เงื่อนไข**: คะแนนจะถูกเครดิตหลังจากเงินฝากได้รับการยืนยันบนบล็อกเชน

### 3. การซื้อขาย (Trading)

เข้าร่วมในตลาดการทำนายผลและรับคะแนนจากกิจกรรมการซื้อขายของคุณ

- **ต่อการซื้อขาย**: รับ **100 คะแนน** สำหรับทุกการซื้อขายที่เสร็จสมบูรณ์ (ไม่ว่าจะเป็นการซื้อหรือขาย)
- **ปริมาณการซื้อขาย**: โบนัสคะแนนสำหรับผู้ที่มีปริมาณการซื้อขายสูง (รายละเอียดจะประกาศให้ทราบภายหลัง)

### 4. การสร้างตลาด (Market Creation)

มีส่วนร่วมในการสร้างเนื้อหาโดยการเสนอและสร้างตลาดใหม่

- **รางวัลการสร้าง**: รับ **2,000 คะแนน** สำหรับแต่ละตลาดที่คุณสร้างและได้รับการอนุมัติ
- **เงื่อนไข**: ตลาดต้องผ่านการตรวจสอบและเปิดให้ซื้อขายได้จริง

## ประโยชน์ของคะแนน

คะแนนของคุณมีค่ามากกว่าที่คุณคิด:

- **สิทธิ์ Airdrop**: คะแนนของคุณจะเป็นปัจจัยหลักในการกำหนดจำนวนโทเค็นที่คุณจะได้รับในการแจก Airdrop รายไตรมาสของเรา
- **สถานะ VIP**: ผู้ที่มีคะแนนสะสมสูงจะได้รับสถานะ VIP พร้อมสิทธิพิเศษต่างๆ เช่น ค่าธรรมเนียมที่ลดลงและการสนับสนุนลำดับความสำคัญ
- **การแลกของรางวัล**: ในอนาคต คุณจะสามารถใช้คะแนนเพื่อแลกรับสินค้าดิจิทัลหรือบริการพิเศษบนแพลตฟอร์ม

## การตรวจสอบคะแนนของคุณ

คุณสามารถตรวจสอบยอดคะแนนสะสมและประวัติการรับคะแนนของคุณได้ที่หน้า "แดชบอร์ด" (Dashboard) ภายใต้ส่วน "รางวัลของฉัน" (My Rewards)
