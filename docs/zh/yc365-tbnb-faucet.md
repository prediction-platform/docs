## 🔧 故障排除

### 常见问题

#### 找不到管理员

**问题**：无法找到或联系管理员

**解决方案**：
- 检查 Discord 服务器角色，寻找版主/管理员
- 在相应频道使用 `@admin` 或 `@moderator` 提及
- 保持耐心 - 管理员可能很忙，请等待回复
- 查看服务器规则，找到请求水龙头的正确频道

#### 社交媒体验证

**问题**：管理员要求提供关注证明

**解决方案**：
- 截图显示您关注了所有三个平台
- 分享您的 Twitter/X 用户名以供验证
- 准备好显示您的 Telegram 和 Discord 成员身份

#### 未收到 tBNB

**问题**：申请了 tBNB 但未收到

**解决方案**：
- 仔细检查您的钱包地址是否正确（BSC 测试网）
- 验证您在钱包中查看的是正确的网络
- 等待 5-10 分钟并刷新您的钱包
- 在 [BSC 测试网浏览器](https://testnet.bscscan.com)上检查交易
- 如果问题持续存在，请再次联系管理员

#### 网络错误

**问题**：连接到错误的网络

**解决方案**：
1. 打开 MetaMask 或您的钱包
2. 点击网络下拉菜单
3. 选择"BSC 测试网"
4. 如果不可用，手动添加 BSC 测试网：
   - 网络名称：BSC Testnet
   - RPC URL：https://data-seed-prebsc-1-s1.binance.org:8545/
   - 链 ID：97
   - 符号：tBNB
   - 浏览器：https://testnet.bscscan.com
