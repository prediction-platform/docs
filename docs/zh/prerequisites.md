# 使用 YC365 的先决条件

## 概述

在开始使用 YC365 之前，您需要满足某些要求并完成一些设置步骤。本指南将帮助您准备参与 YC365 预测市场所需的一切。

## 必要先决条件

### 1. 安装兼容的钱包

YC365 支持所有兼容 EVM 的钱包。您需要安装以下之一：

#### 热门钱包选项：
- **MetaMask** - 最受欢迎且使用最广泛
- **WalletConnect** - 多钱包支持
- **Trust Wallet** - 移动优先钱包
- **Binance Wallet** - 官方币安钱包
- **Coinbase Wallet** - 用户友好的界面
- **任何兼容 EVM 的钱包**

!!! tip "推荐钱包"
    MetaMask 是初学者最广泛支持且用户友好的选项。

#### 安装步骤：
1. 访问您选择的钱包的官方网站
2. 下载并安装钱包扩展/应用程序
3. 按照钱包的设置说明进行操作
4. 创建新钱包或导入现有钱包
5. **重要**：安全存储您的助记词/私钥

### 2. 配置 BSC 主网

您的钱包必须连接到币安智能链 (BSC) 网络才能使用 YC365。

#### 自动配置：
当您访问 dapp.yc365.io 时，大多数钱包会自动检测并添加 BSC 网络。

#### 手动配置（如果需要）：
如果 BSC 网络不可用，请使用以下设置手动添加：

- **网络名称**：BSC Mainnet
- **RPC URL**：https://bsc-dataseed1.binance.org/
- **链 ID**：56
- **货币符号**：BNB
- **区块浏览器**：https://bscscan.com/

!!! warning "网络验证"
    在进行任何交易之前，请务必验证您已连接到 BSC 主网。

### 3. 获取所需资产

您需要两种类型的资产来使用 YC365：

#### USDT (BEP-20)
- **用途**：预测市场的主要交易货币
- **最低**：建议 10 USDT 起步
- **网络**：必须是 BSC 网络上的 BEP-20 代币

#### BNB (币安币)
- **用途**：交易的 Gas 费用
- **最低**：建议 0.01 BNB 用于多次交易
- **网络**：原生 BSC 代币

#### 如何获取这些资产：
1. **从中心化交易所**：
   - 在币安、Coinbase 等购买 USDT 和 BNB
   - 提现到 BSC 网络

2. **从去中心化交易所 (DEX)**：
   - 使用 PancakeSwap 或其他 BSC DEX
   - 将其他代币兑换为 USDT 和 BNB

3. **从其他网络**：
   - 从以太坊、Polygon 等跨链资产
   - 使用官方桥接或第三方服务

!!! info "资产要求"
    - USDT 用于交易预测
    - BNB 用于交易 Gas 费用
    - 两者都必须在 BSC 网络上

## 开始使用 YC365

### 第 1 步：完成先决条件
1. ✅ 安装兼容钱包
2. ✅ 配置 BSC 主网
3. ✅ 获取 USDT 和 BNB 资产

### 第 2 步：访问平台
1. 访问 **dapp.yc365.io**
2. 连接您的钱包
3. 验证您在 BSC 主网上

### 第 3 步：充值资产
1. 点击 **"充值"** 按钮
2. 遵循充值流程
3. 等待确认（5-20 秒）
4. 验证资产出现在您的金库中

### 第 4 步：开始交易
1. 浏览可用的预测市场
2. 进行预测
3. 监控您的头寸

## 资产管理

### 充值
- **流程**：将 USDT 从您的钱包转移到 YC365 金库
- **用途**：为您的交易账户注资
- **时间**：5-20 秒确认

### 提现
- **流程**：将 USDT 从金库转回您的钱包
- **用途**：提取您的奖金或未使用的资金
- **时间**：5-30 秒确认

!!! success "关键点"
    - 充值为您的交易账户注资
    - 提现将资金返还到您的个人钱包
    - 所有交易都需要 BNB 作为 Gas 费用

## 安全注意事项

### 钱包安全
- **切勿分享您的私钥或助记词**
- **大额资金使用硬件钱包**
- **在可用时启用 2FA**
- **连接前验证网站 URL**

### 交易安全
- **始终验证交易详情**
- **确认前检查 Gas 费用**
- **仅使用受信任的网络**
- **保留一些 BNB 用于 Gas 费用**

### 平台安全
- **仅使用官方 dapp.yc365.io**
- **验证智能合约地址**
- **立即报告可疑活动**

## 故障排除

### 常见问题

**问：我的钱包无法连接**
答：确保您使用的是兼容的 EVM 钱包并已正确配置 BSC 主网。

**问：我没有足够的 BNB 支付 Gas 费**
答：从交易所购买 BNB 或在 DEX 上将其他代币兑换为 BNB。

**问：我的 USDT 在错误的网络上**
答：使用桥接服务将 USDT 转移到 BSC 网络，或直接在 BSC 上购买 USDT。

**问：平台无法识别我的资产**
答：确保您的资产在 BSC 网络上，并且您已连接到正确的钱包。
