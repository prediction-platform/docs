## üîß Troubleshooting

### Common Issues

#### Claim Request Failed

**Problem**: Faucet request fails or shows error

**Solutions**:
- Verify you're on BSC Testnet (Chain ID: 97)
- Check you haven't claimed in the last 24 hours
- Ensure your wallet is properly connected
- Try again after a few minutes

#### Tokens Not Received

**Problem**: Claimed tokens but didn't receive them

**Solutions**:
- Wait 30-60 seconds and refresh your wallet
- Check you're viewing the correct network (BSC Testnet)
- Verify the transaction on [BSC Testnet Explorer](https://testnet.bscscan.com)
- Contact support if issue persists

#### Rate Limit Error

**Problem**: "You've already claimed" or rate limit error

**Solutions**:
- Wait for the 24-hour cooldown period
- Check the exact time of your last claim
- Try the alternative faucet channel (dApp or Discord)

#### Wrong Network

**Problem**: Connected to wrong network

**Solutions**:
1. Open MetaMask
2. Click network dropdown
3. Select "BSC Testnet" 
4. If not available, add BSC Testnet manually:
   - Network Name: BSC Testnet
   - RPC URL: https://data-seed-prebsc-1-s1.binance.org:8545/
   - Chain ID: 97
   - Symbol: tBNB
   - Explorer: https://testnet.bscscan.com
