    YC365 Faucet - Get Test Tokens

    Welcome to the YC365 Faucet! This guide will help you obtain test tokens for testing and development on the YC365 platform.

    üìú Faucet Rules

    Claim Limits

    The YC365 Faucet provides test USDT tokens through two channels:

    Channel Amount per Claim Frequency Requirements
    dApp Faucet 50 USDT Once every 24 hours Connected wallet
    Discord Faucet 100 USDT Once every 24 hours Discord member

    !!! note "Important" - Each wallet address can claim once per 24 hours per channel - Test tokens have no real value and are for testing purposes only - Never use test tokens on mainnet

    üö∞ How to Get Test USDT

    Method 1: YC365 dApp Faucet

    1. Connect Your Wallet
    2. Visit dapp.yc365.io
    3. Connect your MetaMask or compatible wallet

    4. Ensure you're on BSC Testnet

    5. Access the Faucet

    6. Navigate to the "Faucet" section

    7. Click "Claim Test USDT"

    8. Receive Tokens

    9. You'll receive 50 test USDT
    10. Tokens will appear in your wallet within seconds
    11. Wait 24 hours before claiming again

    Method 2: Discord Faucet

    1. Join YC365 Discord
    2. Join our official Discord server

    3. Complete server verification

    4. Navigate to Faucet Channel

    5. Go to the #faucet channel

    6. Read the pinned instructions

    7. Request Tokens

    8. Use the faucet bot command with your wallet address
    9. Receive 100 test USDT
    10. Wait 24 hours before requesting again

    üîó Get Test BNB (tBNB)

    To interact with the YC365 platform, you'll also need test BNB (tBNB) for gas fees.

    Get tBNB from BSC Official Faucet

    Visit the official Binance Smart Chain Testnet Faucet:

    üëâ https://www.bnbchain.org/en/testnet-faucet

    Steps to Get tBNB:

    1. Visit the BSC Faucet
    2. Click the link above to open the official BSC testnet faucet

    3. You'll need 0.002 BNB on BSC Mainnet for verification

    4. Enter Your Wallet Address

    5. Paste your BSC testnet wallet address

    6. Select "0.3 BNB" from the dropdown

    7. Submit Request

    8. Click "Send 0.3 BNB"

    9. Wait for the transaction to complete

    10. Verify Receipt

    11. Check your wallet for the test BNB
    12. You can request again after 24 hours

    !!! tip "Alternative tBNB Sources" - BNB Chain Discord: Join the BNB Chain Discord and use their faucet channel - QuickNode: Use QuickNode's BSC faucet - Chainstack: Try Chainstack's faucet

    üîß Troubleshooting

    Common Issues

    Claim Request Failed

    Problem: Faucet request fails or shows error

    Solutions: - Verify you're on BSC Testnet (Chain ID: 97) - Check you haven't claimed in the last 24 hours - Ensure your wallet is properly connected - Try again after a few minutes

    Tokens Not Received

    Problem: Claimed tokens but didn't receive them

    Solutions: - Wait 30-60 seconds and refresh your wallet - Check you're viewing the correct network (BSC Testnet) - Verify the transaction on BSC Testnet Explorer - Contact support if issue persists

    Rate Limit Error

    Problem: "You've already claimed" or rate limit error

    Solutions: - Wait for the 24-hour cooldown period - Check the exact time of your last claim - Try the alternative faucet channel (dApp or Discord)

    Wrong Network

    Problem: Connected to wrong network

    Solutions: 1. Open MetaMask 2. Click network dropdown 3. Select "BSC Testnet" 4. If not available, add BSC Testnet manually: - Network Name: BSC Testnet - RPC URL: https://data-seed-prebsc-1-s1.binance.org:8545/ - Chain ID: 97 - Symbol: tBNB - Explorer: https://testnet.bscscan.com
