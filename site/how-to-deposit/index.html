    How to Deposit

    Overview

    Making your first deposit on YC365 is a straightforward process that enables you to start trading on prediction markets. This comprehensive guide will walk you through every step, from wallet setup to completing your first deposit transaction.

    Prerequisites Checklist

    Before you begin the deposit process, ensure you have:

    ‚úÖ Wallet Requirements

    • EVM-compatible wallet installed (MetaMask recommended)
    • BSC Mainnet configured in your wallet
    • Wallet properly secured with strong password and backup phrase

    ‚úÖ Asset Requirements

    • USDT (BEP-20) tokens for trading
    • BNB tokens for gas fees (minimum 0.01 BNB recommended)
    • Assets in BSC wallet (not on other networks)

    ‚úÖ Technical Requirements

    • Stable internet connection
    • Updated browser (Chrome, Firefox, Safari, or Edge)
    • Access to dapp.yc365.io

    Step-by-Step Deposit Guide

    Step 1: Wallet Setup and Configuration

    Installing MetaMask (Recommended)

    If you don't have MetaMask installed:

    1. Download MetaMask
    2. Visit metamask.io
    3. Click "Download" and select your browser
    4. Install the browser extension

    5. Follow setup instructions to create a new wallet

    6. Secure Your Wallet

    7. Write down your 12-word seed phrase securely
    8. Store it in a safe place (never digital)
    9. Set a strong password
    10. Enable all security features

    Configuring BSC Network

    Add Binance Smart Chain to your MetaMask:

    1. Open MetaMask
    2. Click the network dropdown (usually shows "Ethereum Mainnet")

    3. Select "Add Network" or "Custom RPC"

    4. Enter BSC Network Details Network Name: Smart Chain New RPC URL: https://bsc-dataseed.binance.org/ Chain ID: 56 Currency Symbol: BNB Block Explorer URL: https://bscscan.com

    5. Save and Switch
    6. Click "Save" to add the network
    7. Switch to "Smart Chain" network
    8. Verify you see BNB as the native token

    Step 2: Acquiring Required Assets

    Getting USDT (BEP-20)

    You need USDT on the Binance Smart Chain for trading:

    Option 1: Purchase from Exchange 1. Buy on Binance - Create account on Binance.com - Complete KYC verification - Buy USDT using fiat or crypto - Withdraw to your BSC wallet address

    1. Other Exchanges
    2. KuCoin, Gate.io, or other BSC-supporting exchanges
    3. Ensure you select "BEP-20" network for withdrawal
    4. Double-check wallet address before sending

    Option 2: Swap from Other Tokens 1. Use PancakeSwap - Visit pancakeswap.finance - Connect your wallet - Swap BNB or other tokens for USDT

    1. Use 1inch or Other DEXs
    2. Compare rates across multiple DEXs
    3. Execute swap for best price

    Getting BNB for Gas Fees

    You need BNB to pay for transaction fees:

    1. Minimum Amount
    2. Hold at least 0.01 BNB for gas fees

    3. 0.05 BNB recommended for multiple transactions

    4. Acquisition Methods

    5. Buy directly on exchanges
    6. Receive from faucets (small amounts(https://www.bnbchain.org/en/testnet-faucet))
    7. Swap from other tokens on DEXs

    Step 3: Connecting to YC365

    Accessing the Platform

    1. Visit YC365
    2. Navigate to dapp.yc365.io
    3. Ensure URL is correct (check for https://)

    4. Bookmark the official URL for future use

    5. Connect Your Wallet

    6. Click "Connect Wallet" button
    7. Select "MetaMask" from wallet options
    8. Approve connection in MetaMask popup
    9. Verify wallet address is displayed correctly

    Verifying Connection

    1. Check Network
    2. Ensure you're on BSC network
    3. Platform should show "BSC" or "Smart Chain"

    4. Switch networks if necessary

    5. Check Balance Display

    6. Platform should show your USDT balance
    7. BNB balance should also be visible
    8. Refresh if balances don't appear

    Step 4: Making Your First Deposit

    Accessing Deposit Function

    1. Navigate to Deposit
    2. Click on "Deposit" button or menu item
    3. This may be in the main interface or under "Wallet"

    4. You'll be taken to the deposit interface

    5. Review Deposit Information

    6. Check supported assets (USDT BEP-20)
    7. Review minimum deposit amounts
    8. Understand any deposit fees

    Completing the Deposit

    1. Enter Deposit Amount
    2. Specify amount of USDT to deposit
    3. Start with a small amount for your first deposit

    4. Ensure you have enough BNB for gas fees

    5. Review Transaction Details Asset: USDT (BEP-20) Amount: [Your specified amount] Network Fee: ~0.005-0.02 BNB Destination: YC365 Vault Contract

    6. Confirm in Wallet

    7. Click "Deposit" or "Confirm" button
    8. MetaMask popup will appear
    9. Review all transaction details carefully

    10. Click "Confirm" in MetaMask

    11. Wait for Confirmation

    12. Transaction will be broadcast to BSC
    13. Wait for blockchain confirmation (usually 3-15 seconds)
    14. Do not close browser or refresh page

    Verification and Completion

    1. Check Transaction Status
    2. Platform will show "Pending" then "Confirmed"
    3. You can view transaction on BSCScan

    4. Save transaction hash for records

    5. Verify Balance Update

    6. Your YC365 balance should update automatically
    7. Allow 5-20 seconds for balance refresh
    8. Contact support if balance doesn't update

    Example Deposit Walkthrough

    Let's walk through a typical first deposit:

    Scenario: New user wants to deposit 100 USDT

    Security Best Practices

    üîí Transaction Security

    Before Submitting Transaction

    • Verify Website URL: Always check you're on dapp.yc365.io
    • Check Wallet Connection: Ensure correct wallet is connected
    • Review Amount: Double-check deposit amount is correct
    • Confirm Network: Verify you're on BSC network

    During Transaction

    • Don't Rush: Take time to review all details
    • Check Gas Fee: Ensure gas fee is reasonable
    • Verify Contract: Transaction should be to YC365 vault contract
    • Stay Connected: Keep browser open until confirmation

    After Transaction

    • Save Records: Screenshot or save transaction hash
    • Monitor Balance: Wait for balance update
    • Verify on Explorer: Check transaction on BSCScan
    • Report Issues: Contact support if problems occur

    üõ°Ô∏è Wallet Security

    MetaMask Security

    • Lock When Not Using: Always lock MetaMask when done
    • Regular Updates: Keep MetaMask updated to latest version
    • Phishing Protection: Never enter seed phrase on websites
    • Hardware Wallet: Consider hardware wallet for large amounts

    Private Key Protection

    • Never Share: Never share private keys or seed phrases
    • Secure Storage: Store seed phrase offline and securely
    • Multiple Copies: Keep backup copies in separate locations
    • No Digital Storage: Avoid storing seed phrases digitally

    Common Issues and Solutions

    üîß Transaction Issues

    Transaction Failed

    Problem: Deposit transaction failed or reverted

    Possible Causes: - Insufficient BNB for gas fees - Network congestion - Wrong network selected - Contract interaction error

    Solutions: 1. Check BNB balance and add more if needed 2. Wait for lower network congestion 3. Verify you're on BSC network 4. Try transaction again with higher gas fee 5. Contact support if issue persists

    Transaction Stuck/Pending

    Problem: Transaction shows as pending for long time

    Solutions: 1. Wait longer (BSC can occasionally have delays) 2. Check BSCScan for transaction status 3. Try speeding up transaction in MetaMask 4. Cancel and retry if necessary

    Balance Not Updated

    Problem: Deposited funds not showing in account

    Solutions: 1. Wait 30 seconds and refresh page 2. Check transaction was confirmed on BSCScan 3. Verify transaction was to correct contract 4. Contact support with transaction hash

    üîß Wallet Issues

    Cannot Connect Wallet

    Problem: MetaMask won't connect to platform

    Solutions: 1. Refresh browser page 2. Unlock MetaMask 3. Switch to BSC network 4. Clear browser cache 5. Try different browser 6. Restart MetaMask

    Wrong Network

    Problem: Wallet connected to wrong network

    Solutions: 1. Open MetaMask 2. Click network dropdown 3. Select "Smart Chain" or "BSC" 4. Refresh platform page

    üîß Asset Issues

    No USDT Balance

    Problem: Don't have USDT to deposit

    Solutions: 1. Purchase USDT on exchange and withdraw to BSC 2. Swap other tokens for USDT on PancakeSwap 3. Use cross-chain bridge if you have USDT on other networks

    Insufficient Gas

    Problem: Not enough BNB for transaction fees

    Solutions: 1. Buy small amount of BNB on exchange 2. Use BSC faucet for minimal amounts 3. Ask someone to send small amount of BNB
